# PROJETO MOVIEFLIX

Projeto de exemplo criado em aula para tumar do JornadaDev RS.

## Sobre o projeto

Projeto desenvolvido para front-end, simulando um stream de filmes. O designe da aplicação foi inspirado na Netflix.

## Tecnologias utilizadas

- **React**: Biblioteca para construção da interface.
- **Vite**: Ferramenta para build e desenvolvimento.
- **Javascript**: Linguagem principal.
- **React Router DOM**: Controle de rotas e navegação.
- **JSON**: Mock de dados.

## Instalação e execução

- Pré-requisitos: Node.js e NPM;
- Clone o repositório;
- Instale as dependencia do projeto (npm install);
- Execute o projeto (npm run dev).
